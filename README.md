### 🌟 **Engineering Impact & Featured Projects**

#### 🛡️ **Khadmaty - Local Service Marketplace** *(Dec 2025 – Present)*
> A full-stack marketplace connecting customers with local service providers. This project showcases my deep understanding of scalable architectures.
- **Architecture:** Implemented **Clean Architecture** with a strict 3-layer separation (Data, Domain, Presentation), ensuring robust maintainability and testability.
- **Scalability:** Built a comprehensive **Repository Pattern** with abstract interfaces for flexible backend switching and configured **Dependency Injection** using GetIt with 10+ registered services.
- **Features:** Integrated Firestore real-time streams, developed dual service types (On-demand/Appointment), and ensured full **RTL & Localization** support with responsive layouts for Mobile, Tablet, and Desktop.

#### 🎓 **Rozewell Learning Center**
> A real-time student-teacher communication app designed for a large educational institution, currently serving **1000+ active users**.
- **Performance:** Significantly reduced notification delivery time by implementing **Firebase Cloud Messaging** and optimizing 10+ REST API endpoints.
- **Success:** Achieved a **4.2+ star rating** on the Google Play Store, reflecting high user satisfaction and stability.
- **Link:** [View on Google Play](https://play.google.com/store/apps/details?id=at.rozewail.rozewailapp)

#### 🧠 **Hcc_Detector (AI-Powered Desktop Application)**
> An innovative AI-powered desktop application developed to assist medical professionals in diagnosing Hepatocellular Carcinoma (HCC).
- **AI Core:** Integrated a sophisticated backend powered by a **PyTorch deep learning model** meticulously trained on real CT images from Al-Rajhi Liver Hospital.
- **Utility:** Provides crucial diagnostic support to doctors, particularly in identifying HCC cases where traditional CT image analysis might be inconclusive.
